# Card Deck Spec

LumenSword369 includes a printable sheathed-sword card deck.

## Front card fields

- Card number
- Possibility pattern
- Sheathed sword sentence
- Purpose
- Reflection prompt

## Back card identity

- LS369 sigil
- The sword cuts fog, not people.
- Truth · Mercy · Consent · Possibility
- Validate · Empower · Clarify · Invite
- Use language to strengthen agency, never to override it.

## Design principle

The deck should feel like disciplined clarity, not aggression. The sword is symbolic: available power under ethical restraint.
