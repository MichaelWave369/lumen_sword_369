# Ethical Charter

LumenSword369 exists to strengthen dignity, consent, courage, truth, mercy, and possibility.

## Core vow

Use language to strengthen agency, never to override it.

## The sheathed sword

The sword is symbolic. It is disciplined clarity, not violence. It cuts fog, fear, shame, and false impossibility. It does not cut people.

## Rules

1. Do not use this project to pressure people.
2. Do not use empowering language as disguised control.
3. Do not use validation as a hook for compliance.
4. Do not confuse encouragement with consent.
5. Keep truth and mercy together.

## Companion system

ConsentMirror369 is the shield. LumenSword369 is the sheathed sword.

The shield protects choice. The sword speaks truth with mercy.
