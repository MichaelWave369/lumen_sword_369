# Scenario Examples

LumenSword369 includes real-world practice scenarios so users can learn empowering speech through context.

## Current categories

- Family
- Faith
- Workplace
- Grief
- Conflict
- Apology

## Scenario structure

Each scenario includes:

- `id`
- `category`
- `title`
- `situation`
- `starting_sentence`
- `possible_patterns`
- `reflection_questions`
- `empowering_response`

## Design principle

Scenarios should strengthen agency, dignity, and possibility. They should not coach users to overpower, guilt, shame, or corner anyone.

## Safety reminder

If direct conversation could increase danger, do not rely on a script. Prioritize trusted support, distance, and qualified help.
