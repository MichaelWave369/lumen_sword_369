# Security and Safety Policy

LumenSword369 is a local-first empowering-speech and possibility-language project.

## Supported version

| Version | Supported |
|---|---|
| 1.0.x | Yes, once released |

## Reporting security concerns

If you find a security issue, please open a GitHub issue with enough detail to reproduce the concern, unless doing so would expose private user data or create risk for others.

For sensitive issues, contact the maintainer directly through the contact information listed on the GitHub profile.

## Safety concerns

LumenSword369 should not be used to pressure, shame, coerce, manipulate, love-bomb, threaten, or overpower people.

If you notice content that appears to teach harmful persuasion, disguised coercion, intimidation, retaliation, or emotional control, please report it as a safety concern.

## Privacy model

The app stores practice entries in browser local storage. It does not send those entries to a server by default. Local storage is not encrypted, so anyone with access to the same browser profile may be able to view saved entries.

## Emergency disclaimer

LumenSword369 is not an emergency service. If someone is in immediate danger, they should contact trusted support, local emergency services, or qualified professional help.
