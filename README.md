# LumenSword369

**LumenSword369 is a humane empowering-speech and possibility-language project.**

It is the companion to ConsentMirror369. If ConsentMirror369 is the shield that helps people recognize pressure, LumenSword369 is the sheathed sword: disciplined clarity that strengthens dignity, consent, courage, and possibility without overpowering anyone.

> The sword cuts fog, not people.

## Current status

v0.2 prototype.

## Core vow

Use language to strengthen agency, never to override it.

## What this project is

- A positive conversation literacy framework
- A collection of empowering sentences
- A consent-first response practice tool
- A local-first reflection app prototype
- A printable card deck concept

## What this project is not

- Not a persuasion weapon
- Not manipulation with nicer words
- Not love-bombing
- Not a way to win conversations
- Not a replacement for therapy, legal advice, or crisis support

## What is included

- React and Vite app
- 17 Sheathed Sword Sentences
- Possibility pattern taxonomy
- Searchable sentence browser
- Selected sentence detail view
- Local-only practice log
- Practice log JSON export/import
- Duplicate-aware import handling
- Printable sheathed-sword card deck
- Ethics, safety, privacy, and card deck docs
- GitHub Pages deployment workflow
- Verification workflow

## Core method

1. **Validate** — Let the person know their experience matters.
2. **Empower** — Restore choice and agency.
3. **Clarify** — Separate truth from fear, shame, or fog.
4. **Invite** — Open a next step without forcing it.

## Starter sentence

> I want your real answer, not your pressured answer.

## Privacy model

The practice log is local-first. Entries are saved only in the user's browser through local storage. The app includes export, import, duplicate-skip, and clear controls so the user can keep, restore, or remove their own practice entries.

Local storage is not encrypted. Anyone with access to the same browser profile may be able to view saved entries.

## Project boundaries

LumenSword369 is an education and reflection project. It does not guarantee emotional outcomes, make people agree, diagnose relationships, replace therapy, replace legal advice, or guarantee safety.

Empowering language must not become disguised pressure.

If there is immediate danger, threats, stalking, coercive control, self-harm risk, or fear for physical safety, prioritize trusted support, local emergency services, or qualified professional help.
